📊 Dashboard de Reclamações e Relacionamento com Clientes
📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de transformar dados de reclamações de clientes em informações estratégicas para apoio à tomada de decisão.

A proposta consiste na criação de um dashboard gerencial capaz de monitorar indicadores de desempenho relacionados ao atendimento, resolução de problemas e satisfação dos clientes.

Por meio da análise dos dados, é possível identificar padrões, causas recorrentes, áreas críticas e oportunidades de melhoria nos processos internos da organização.

🎯 Objetivos
Monitorar o volume de reclamações.
Identificar as principais causas dos problemas reportados.
Avaliar a eficiência do processo de resolução.
Detectar clientes reincidentes.
Apoiar decisões baseadas em dados.
Melhorar a qualidade do atendimento ao cliente.
📈 Indicadores (KPIs)

O dashboard apresenta os seguintes indicadores:

Total de Reclamações

Quantidade total de registros analisados.

Taxa de Resolução

Percentual de reclamações solucionadas em relação ao total registrado.

Tempo Médio de Resolução

Tempo médio necessário para conclusão dos atendimentos.

Clientes Reincidentes

Quantidade de clientes que registraram mais de uma reclamação.

Casos Críticos

Ocorrências classificadas como prioritárias ou de alto impacto.

📊 Visualizações Desenvolvidas
Evolução Mensal das Reclamações

Permite acompanhar tendências e identificar períodos de aumento ou redução de ocorrências.

Gráfico de Pareto

Utilizado para identificar as causas que mais contribuem para o volume total de reclamações.

Reclamações por Área Responsável

Mostra quais setores concentram a maior quantidade de demandas.

Status das Reclamações

Distribuição entre:

Em andamento
Resolvidas
Pendentes
Mapa de Calor

Identifica períodos com maior concentração de ocorrências.
